# Seagrass Guardian

Seagrass Guardian is an interactive educational web platform designed to educate students about the critical role of seagrass ecosystems in Southeast Asia.

## Features

- **Hero Section**: Interactive map highlighting seagrass hotspots.
- **Blue Carbon Calculator**: Calculate the carbon storage potential of seagrass meadows.
- **Virtual Quadrat Sampling**: Simulates scientific monitoring protocols.
- **Dugong Survival Timeline**: Interactive chart showing the ecological crisis.
- **Restoration Sim**: Decision-tree simulation for meadow restoration.

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React

## Setup Instructions

Prerequisites: Node.js (v18+ recommended)

1.  Clone the repository or navigate to the project directory.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## Notes

- This project uses Tailwind CSS for styling.
- Ensure you have an internet connection to load the font assets and placeholder images.
